# UI Components Module

## Purpose
Reusable UI widget library for consistent look and feel across all screens.

## Responsibilities
- Panel drawing with title bars
- Button rendering with hover states
- Labels and value displays
- Slider controls
- Icon/progress bars
- Hit-testing utilities
- Number formatting

## Public API
See `ui.components` facade.

## Dependencies
- ui.theme

## Used by
- All UI screens and dialogs
