# Settings Module

## Purpose
Settings screen for configuring language, UI style, audio, and display options.

## Responsibilities
- Language selection (DE/EN/ZH)
- UI style switching (retro/clean)
- Volume sliders (master, music, SFX)
- FPS counter toggle
- Fullscreen toggle
- Settings persistence

## Public API
See `ui.settings` facade.

## Dependencies
- ui.components
- core (config, translator, eventbus)
