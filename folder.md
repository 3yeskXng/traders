# Traders Project Root

## Purpose
Root directory of the Traders trading simulation game.

## Directories
- `core/` - Core engine (eventbus, config, logging, json, fonts, utilities)
- `simulation/` - Game logic (no LÖVE dependency)
- `rendering/` - Graphics output using LÖVE
- `ui/` - User interface screens and components
- `savegame/` - Save/Load system
- `mods/` - Mod loading infrastructure
- `multiplayer/` - Multiplayer placeholder
- `data/` - JSON data files (cities, goods, ships, etc.)
- `assets/` - Resource files (images, sounds, fonts)
- `tests/` - Test scripts

## Entry Point
`main.lua` - LÖVE game entry point
