# Economy Module

## Purpose
Economic simulation including prices, demand, supply, production, consumption, and inflation.

## Public API
- `simulation.economy.prices` - Price calculations (updateCityPrices)
- `simulation.economy.demand` - Demand calculations
- `simulation.economy.supply` - Supply calculations
- `simulation.economy.production` - Production simulation
- `simulation.economy.consumption` - Consumption simulation
- `simulation.economy.inflation` - Inflation mechanics

## Dependencies
- core.utils

## Used by
- simulation.world.bootstrap
